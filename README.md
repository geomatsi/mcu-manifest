# mcu-manifest
Manifests for MCU projects

## stm32f4
$ mkdir project  
$ cd project  
$ repo init -u git@github.com:geomatsi/mcu-manifest.git -b master -m stm32-manifest.xml  
$ repo sync -c  

## avr
